# 1HandIndia Brand Direction

## 1. Brand Name

**Selected working brand name:** 1HandIndia

1HandIndia should feel like a serious Indian marketplace brand: simple to say, easy to remember, and suitable for customer shopping, local sellers, wholesale distributors, and B2B buyers.

## 2. Logo Status

The logo is pending and can be designed later.

When designing the logo, keep it:

- Clean and readable at small mobile-app sizes.
- Strong enough for website header, app icon, invoice, seller dashboard, and social media.
- Original, not visually copied from Flipkart, Amazon, Meesho, IndiaMART, or any existing ecommerce brand.

## 3. Recommended Elegant Color Palette

| Purpose | Color Name | Hex |
|---|---|---|
| Primary brand | Royal Navy | `#163B5C` |
| Main action / offer accent | Saffron Orange | `#F57C00` |
| Trust / success / seller growth | Marketplace Green | `#0F8A5F` |
| Premium highlight | Soft Gold | `#D9A441` |
| Page background | Warm Ivory | `#FAF7F0` |
| Main text | Charcoal Ink | `#1F2933` |
| Border / quiet UI | Mist Grey | `#E5E7EB` |
| Error / destructive action | Alert Red | `#D64545` |

## 4. UI Color Usage

- Use **Royal Navy** for header, footer, important navigation, admin/seller control surfaces, and trust-heavy sections.
- Use **Saffron Orange** for primary CTA buttons, sale labels, highlighted offers, and important customer actions.
- Use **Marketplace Green** for success states, verified seller indicators, payout completion, delivery completion, and positive trust signals.
- Use **Warm Ivory** as a soft page background where a lighter premium feel is needed.
- Use **Charcoal Ink** for body text and dashboard readability.
- Use **Mist Grey** for cards, borders, tables, form fields, and admin separation lines.

## 5. Brand Feel

1HandIndia should look:

- Indian but modern.
- Trustworthy for payments and sellers.
- Friendly enough for customers.
- Professional enough for sellers, B2B buyers, and admins.
- Premium without looking expensive or complicated.
